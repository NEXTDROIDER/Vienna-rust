# Vienna-rust
Version: 0.0.4

Vienna – your all-in-one tool. Simple to build and run on any system.

---

## 👍 How to Use
use it in windows
```batch
server-portable\ViennaLauncher.exe
```
## 🛠 How to Build

Build from source using Rust:

```batch
build-portable.bat
```
or go ![releases](https://github.com/NEXTDROIDER/Vienna-rust/releases)
